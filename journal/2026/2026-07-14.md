## What I Learned

Today I learned that program performance is influenced by the entire computing stack rather than a single optimization technique.

I studied several software optimization methods, including loop invariant code motion, reducing memory access, strength reduction, loop unrolling and eliminating dependencies.

I also explored how modern CPUs execute instructions through pipelining, branch prediction and out-of-order execution.

---

## Biggest Insight

Performance optimization is a systems problem.

Algorithms, compilers, CPU architecture, cache hierarchy and operating systems all work together to determine the final execution speed.
