## What I Learned

Today I studied the complete memory hierarchy.

I learned that different storage technologies exist because no single device can simultaneously provide high speed, large capacity and low cost.

I also learned the differences between SRAM, DRAM and Flash memory.

Another interesting topic was how SSDs internally manage Flash memory through the Flash Translation Layer (FTL).

---

## Biggest Insight

One of today's biggest realizations is that every level of the memory hierarchy is a compromise.

Registers are extremely fast but tiny.

Caches improve CPU performance.

DRAM provides large capacity.

SSDs provide persistence.

Each layer hides the limitations of the layer below it.
